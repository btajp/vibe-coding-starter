# ファイル、ディレクトリ、パスを読む

## この章でできるようになること

自分がどの場所で作業しているのか、教材リポジトリがどこにあるのか、パスが何を表しているのかを説明できるようになります。

## 扱うこと

- ファイル、フォルダ、ディレクトリ、拡張子
- パス、絶対パス、相対パス
- ホームディレクトリ、`~`、`.`、`..`
- ユーザー名、ホームディレクトリ、作業ディレクトリ
- `~/src/github.com/btajp/vibe-coding-starter` が何を表しているか
- 教材リポジトリと成果物リポジトリを分ける理由
