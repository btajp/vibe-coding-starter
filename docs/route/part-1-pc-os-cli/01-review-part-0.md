# 第0部で何をしたのかを棚卸しする

## この章でできるようになること

第0部で実行したコマンドを見返し、何をインストールしたのか、どの設定ファイルを変えたのか、どのディレクトリに移動したのかを整理できるようになります。

## 扱うこと

- 第0部で実行したコマンドの一覧を見返す
- インストールしたものと、設定したものを分ける
- 変更された可能性がある場所を整理する
- まだ理解できていない単語をメモする
- AIに説明させる前に、自分で状況を言葉にする
