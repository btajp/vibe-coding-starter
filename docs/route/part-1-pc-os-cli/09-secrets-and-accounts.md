# 秘密情報とログイン状態を混同しない

## この章でできるようになること

ローカルPCのユーザー、GitHubアカウント、AIツールのログインを区別し、AIに貼ってはいけない情報を判断できるようになります。

## 扱うこと

- ローカルPCのユーザー
- GitHubアカウント
- CodexやClaude Codeのログイン
- パスワード
- 認証コード
- トークン
- APIキー
- 秘密鍵
- エラー相談で貼ってよい情報と貼ってはいけない情報
