# ターミナル、CLI、シェル、基本コマンドを読む

## この章でできるようになること

ターミナルに入力するコマンドを、丸暗記ではなく、コマンド本体、オプション、引数、表示結果に分けて読めるようになります。

## 扱うこと

- ターミナル、CLI、シェル
- zshとbash
- `pwd`、`ls`、`cd`、`mkdir`
- コマンド本体、オプション、引数
- 標準出力とエラー表示
- `--version` で確認する意味
- `command not found`、`Permission denied`、`No such file or directory` の見方
