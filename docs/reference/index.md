# リファレンス

リファレンスは、本編を進めるために必須ではない補足資料です。

本編では必要な範囲だけを説明します。
深い背景、比較、トラブル対応、発展的な選択肢は、このリファレンスに置きます。

読まなくても本編は進められます。
ただし、読むと用語や判断の理由を理解しやすくなります。

## 使い方

- 本編で知らない用語が出てきたら、用語集を見る
- コマンドの意味を確認したいときは、コマンド早見表を見る
- エラーで止まったら、トラブルシューティングを見る
- macOSとWSL Ubuntuの違いが気になったら、OS関連の補足を見る
- 公開や運用の判断に迷ったら、安全・運用の補足を見る
