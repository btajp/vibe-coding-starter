# Reviews

このディレクトリは、学習者がこの教材リポジトリへ感想Pull Requestを送るための場所です。

追加する場合は、次の形にしてください。

```text
reviews/YOUR_GITHUB_USERNAME.md
```

書いてよいこと:

- この教材で学んだこと
- つまずいたところ
- AIエージェントを使って気づいたこと
- 今後学びたいこと

書かないこと:

- 本名、住所、電話番号
- 学校名、勤務先など公開したくない所属
- メールアドレス
- APIキー、トークン、秘密鍵、認証コード
- 他人を攻撃する内容

Pull Requestは公開されます。
AIが生成した文章でも、公開してよい内容か自分で確認してから送ってください。
