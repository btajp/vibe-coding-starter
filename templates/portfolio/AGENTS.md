# AGENTS.md

このリポジトリは、学習者のVibe Codingポートフォリオです。

学習者に見える説明は日本語で書いてください。

## 目的

- 最小の静的ポートフォリオページを作る
- 学習ログを残す
- GitHub Pagesで公開する
- 学習者が各変更を理解できるようにする

## ルール

ファイルを編集する前に、次を説明してください。

1. どのファイルを変更するか
2. なぜ変更するか
3. どう確認するか

含めてはいけないもの:

- APIキー
- トークン
- パスワード
- 業務上公開してはいけない情報
- 公開すべきでない個人情報

commit前に、学習者へ次を確認するよう促してください。

```bash
git status
git diff
```

公開前に、GitHub Pagesで誰に何が見えるかをレビューしてください。
