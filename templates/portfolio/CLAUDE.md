# CLAUDE.md

このファイルは、このポートフォリオリポジトリでClaude Codeに作業方針を伝えるためのものです。

## リポジトリの目的

このリポジトリは、学習者のVibe Codingポートフォリオです。

GitHub Pagesで公開する静的Webページを含みます。

## 作業方針

- すべて日本語で説明してください。
- 変更前に、変更対象ファイル、変更理由、確認方法を説明してください。
- 学習者が理解できるように、HTML / CSS / JavaScriptの変更内容を説明してください。
- 秘密情報、個人情報、業務上公開してはいけない情報を含めないでください。
- GitHub Pagesで公開されることを前提にレビューしてください。

## 安全確認

commit前に、学習者に次を促してください。

```text
git status
git diff
```

公開前に、次を確認してください。

- 個人情報がないか
- 秘密情報がないか
- 業務上の機密がないか
- 外部リンクが適切か
- ページ内容が学習ポートフォリオとして自然か
