# My Vibe Coding Portfolio

このリポジトリは、私のVibe Coding学習ポートフォリオです。

## このリポジトリの目的

このリポジトリでは、Vibe Codingを通じて学んだことを記録し、GitHub Pagesで公開します。

## 含める内容

- 自己紹介
- Vibe Codingを学ぶ理由
- 学んだこと
- AIとの協働ルール
- 作ってみたいもの
- 業務改善アイデア
- 学習ログ
- GitHubリポジトリと公開ページへのリンク

```text
  learning-log.md
```

## 公開前の確認

- 個人情報を入れていないか
- 業務上公開してはいけない情報を入れていないか
- APIキーやトークンを入れていないか
- AIが生成した内容を自分で確認したか
- READMEが最低限の説明になっているか
- `YOUR_GITHUB_ID` を自分のGitHubユーザー名に置き換えたか

## 更新の流れ

```bash
git status
git diff
git add .
git commit -m "docs: update portfolio"
git push
```

commit前に、公開してよい内容だけが含まれているか確認します。
